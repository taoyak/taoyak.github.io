python列表函数&&方法
cmp():
cmp(list_1,list_2)
list_1:第一个需要进行比较的列表
list_2:第二个需要进行比较的列表

返回的结果：
如果比较的两个列表是同一类型的元素，则两个列表的值进行比较，并且会返回一个结果。
假如两个列表中的元素不同一类型，则检查它们是否是数字。
如果是数字,执行必要的数字强制类型转换,然后比较。
如果有一方的元素是数字,则另一方的元素”大”(数字是”最小的”)
否则,通过类型名字的字母顺序进行比较。

list_1=[1,2,3]
list_2=(4,5,6)
cmp(list_1,list_2)
-1
cmp(list_2,list_1)
1
list_2=(1,2,3)
cmp(list_1,list_2)
0

max(list_1)
